# Oogway

Medical Records App that combines features from:
- Apple Health App
- GenieMD
- MTBC PHR
- MyChart
- My Medical
- Care Zone
- Capsule PHR

It uses SQLite 3 for storage and uses GPG for encryption.

# Technologies Used

Client:
- C++
- Qt
- GPG
- SQLite 3
- libcurl

Server:
- Node.js
- Express
- MongoDB
